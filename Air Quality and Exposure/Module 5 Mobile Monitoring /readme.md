#__Module 5: Mobile Monitoring__
This module contains basic concepts of mobile monitoring. It explains mobile monitoring approaches including experimental set-up and data processing. 
Sub-topics: Mobile monitoring approaches and analysis: where to use, added value, different approaches, how to set up, data processing, examples.  

Course Detail: The course provides an overview of different mobile monitoring approaches and applications. It will give more detail on the use of portable mid-range instruments (airQmap). It will discuss topics like campaign set-up, data processing, rescaling data and will show some case studies.  

Learning Outcomes:  

Where to use mobile mapping where does it add what are the application areas  

How to set up a mobile mapping campaign what are the requirements, number of repetition, instruments etc...  

Data processing of mobile data How to process the data   

Interpretation of mobile maps how can they be used to provide evidence-based policy recommendations.
