# __Module 5: Mobile Monitoring__
This module is a part of 5 AQE - Air Quality and Exposure 

## __Course details__
This module contains basic concepts of mobile monitoring and provides an overview of different mobile monitoring approaches and applications. It will give more detail on the use of portable mid-range instruments (airQmap). It explains mobile monitoring approaches and will discuss topics like campaign set-up, data processing, rescaling data and will show some case studies.
- Duration: 2 sessions; approx. 1 hrs each 

## __Sub-topics__
- Mobile monitoring applictions and set-up
- Data processing of mobile monitoring and interpretation
- Case studies
- (to be decide: example in field work)
  
## __Learning Outcomes__
- Where to use mobile mapping where does it add and what are the application areas
- How to set up a mobile mapping campaign what are the requirements, number of repetition, instruments etc...
- Data processing of mobile data
- Interpretation of mobile maps how can they be used to provide evidence-based policy recommendations
