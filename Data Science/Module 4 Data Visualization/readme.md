# Module 5: Data Visualization - Python Session

Welcome to the Python part of **Module 5: Data Visualization**.

These materials were prepared for the **first session** of Module 5. The session is intended for learners with little or no prior experience in data science, so the notebooks use simple explanations, step-by-step examples, and guided practice tasks. The emphasis is on clear, practical examples based on environmental and pollutant-style datasets, with a focus on building confidence in reading and creating common scientific plots.

## Notebook Overview

### 1. [Static Plotting with Matplotlib and Seaborn](./1%20Static%20Plotting%20with%20Matplotlib%20and%20Seaborn.ipynb)

This notebook introduces the basics of static plotting in Python. Learners create line plots and bar plots, improve plot readability with titles, labels and legends, and work with simple time-based environmental data.

Main topics:
- plotting with Matplotlib and Seaborn
- line plots and bar plots
- plot titles, axis labels and legends
- improving readability of figures
- plotting temporal environmental data
- saving figures

### 2. [Distributions and Relationships](./2%20Distributions%20and%20Relationships.ipynb)

This notebook focuses on understanding data through visual summaries. Learners explore how values are distributed and how variables relate to one another using standard scientific plots.

Main topics:
- histograms
- boxplots
- scatter plots
- grouped bar plots
- correlation heatmaps
- matrix-style heatmaps

## Suggested Teaching Plan

These materials are intended for **one session of approximately 2 hours**.

Suggested flow:
- **Notebook 1:** 50–60 minutes
- **Notebook 2:** 55–65 minutes

Each notebook includes explanatory text, runnable code examples, guided practice tasks, and a short summary section.

## Learning Outcomes

By the end of this session, learners should be able to:
- create clear static plots to represent temporal trends and environmental variables,
- interpret statistical distributions using histograms and boxplots,
- analyze relationships between continuous variables using scatter plots and correlation heatmaps,
- compare grouped values using bar plots and heatmaps.

## How to Use These Notebooks

1. Upload the files to the appropriate GitHub folder in the E-Learning Platform repository.
2. Open the notebooks in GitHub or Google Colab.
3. Run the cells in order and encourage learners to modify the examples.
4. Use the guided tasks for hands-on classroom work or self-study.
